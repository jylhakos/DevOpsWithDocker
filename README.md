# DevOps with Docker

### Create new Github repository on the command line

$ echo "# DevOps with Docker" >> README.md
$ git init
$ git add README.md
$ git commit -m "first commit"
$ git branch -M main
$ git remote add origin https://github.com/<REPOSITORY>.git
$ git push -u origin main
